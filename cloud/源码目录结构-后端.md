# 源码目录结构解析

## 下表中 `*talbe*` 表示和数据表名一一对应的文件或文件夹

```shell
TDP Cloud
│
├─api ------------------------------------ HTTP 接口
│  ├─alibaba ----------------------------- 阿里云代理
│  ├─certbot ----------------------------- 域名证书接口
│  ├─cloudflare -------------------------- Cloudflare 代理
│  ├─crontab ----------------------------- 计划任务接口
│  ├─tencent ----------------------------- 腾讯云代理
│  ├─terminal ---------------------------- SSH 代理及控制器
│  ├─upgrade ----------------------------- 升级检测服务
│  ├─workhub ----------------------------- 节点服务及控制器
│  └─*talbe* ----------------------------- *talbe*模型控制器
│
├─cmd ------------------------------------ CLI 命令
│  ├─args -------------------------------- 共享参数表
│  ├─initd ------------------------------- 模块初始化
│  └─subset ------------------------------ 子命令集合
│
├─docs ----------------------------------- 说明文档
│
├─docker --------------------------------- Docker 构建工具
│
├─front ---------------------------------- 前端界面
│
├─module --------------------------------- 内部模块
│  ├─certbot ----------------------------- 域名证书服务
│  ├─crontab ----------------------------- 计划任务服务
│  ├─httpd ------------------------------- HTTP 服务器
│  ├─midware ----------------------------- HTTP 中间件
│  ├─migrator ---------------------------- 版本自动迁移
│  ├─upload ------------------------------ 文件上传服务
│  ├─model ------------------------------- 数据库操作封装
│  │  ├─passport ------------------------- 用户中心模型
│  │  └─*talbe* -------------------------- *talbe*表模型
│  ├─worker ------------------------------ 子节点服务
│  └─workhub ----------------------------- 中心节点服务
│
├─service -------------------------------- 服务管理
│  ├─server ------------------------------ 中心节点服务管理
│  └─worker ------------------------------ 子节点服务管理
│
└─tests ---------------------------------- 接口测试
```
