## 土豆片开源

**Open TDP** 是一个源自于腾讯云开发者先锋（TDP）的自由开源技术组织，首批开发者及用户均来自于 TDP 群。我们致力于提供云端前沿应用，目前已经开发了多云资源管理系统`TDP-CLoud`和智能绘画接口平台`TDP-Aiart`。
